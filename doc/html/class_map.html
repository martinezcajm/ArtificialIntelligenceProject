◆ loadMap()

s16 Map::loadMap ( const char *  src,
const char *  background 
)

Loads the map.

Tries to load a map with an image source for the background and an image source for the collisions ( white = free, black = occupied). If another map was already loaded it will be freed. The possible results of this operation are: kErrorCode_InvalidPointer -> The source or the background were nullptr kErrorCode_Memory -> The program was unable of allocating memory kErrorCode_Ok -> Everything went fine

Parameters
srcx coordinate of the position we want to check
backgroundy coordinate of the position we want to check

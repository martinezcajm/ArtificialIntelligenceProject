Detailed Description

Path class.

Class in charge of storing sets of points, it also stores rules to indicate how we want to traverse those points.

◆ setToReady()

s16 Path::setToReady ( )

sets the path to a ready state

Checks that everything was correctly initialized, if that's the case sets the path to ready and from that point the path can be used. Please check the error code to know more info about the reason why the operation fails. kErrorCode_EmptyPath -> The path is empty. kErrorCode_PathNotCreated -> The path has not been created. Make sure you called create() before setToReady kErrorCode_BadLoopsSetting -> The loop setting has not been correctly configured. kErrorCode_BadDirectionSetting -> The direction setting has not been correctly configured.

Returns
s16 result of the operation
